Como borrar partes de fotografías de manera inteligente desde Gimp

Una extensión muy interesante de Gimp es Gimp Resynthesizer, que trata de borrar partes sobrantes de una fotografía de manera inteligente, de manera que no se note el cambio. El resultado puede llegar a ser espectacular: Y como conseguimos llegar a hacer esto? Pues muy fácil, hacemos un yum install ( o apt-get install si estás en ubuntu o derivados) e instalamos: 

gimp-resynthesizer

Ahora abrimos la imagen que queramos, seleccionamos la parte a eliminar (no hace falta ser muy precisos) vamos a filtros -> Realzar -> Smart Remove Selection. Luego pulsamos ok y esperamos… Y magia! :D Yo lo he probado con esta foto: Antes: Después:
