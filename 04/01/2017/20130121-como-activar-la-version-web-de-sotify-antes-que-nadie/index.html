Como activar la version web de Spotify antes que nadie

La beta de Spotify web lleva operativa desde hace algún tiempo desde la web play.spotify.com . Pero si entras directamente, te dirán muy amablemente que te bajes la versión de escritorio. Sin embargo, hay un pequeño truco para colarte y probar esta versión, que de beta tiene poco, pues funciona perfectamente. Lo único que tienes que hacer es acceder a través de un link de facebook, como este: https://apps.facebook.com/get-spotify/?fb_source=notification&fb Una vez cargado, tendrás acceso total a toda la música sin bajarte nada :D
