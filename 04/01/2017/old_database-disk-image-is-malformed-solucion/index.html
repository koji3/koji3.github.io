sqlite> .mode insert

Por último, generamos otra base de datos con la información recopilada: 

sqlite3 database_2.db 
 sqlite> .read dump.sql
 sqlite> .exit

No podemos olvidar que estamos rescatando datos de un archivo corrupto, por lo que es posible que algunas tuplas se pierdan, pero siempre es mejor recuperar algo que nada.
