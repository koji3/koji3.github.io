Instalar Gnome 3 en Ubuntu

El entorno de escritorio de Ubuntu es una basura, todo hay que decirlo. No cabe duda de que unity acabará siendo algo grande, pero ahora mismo no supera a Gnome ni KDE, según mi opinión, así que os voy a enseñar a instalar Gnome3 en Ubuntu. Es muy sencillo, abrís un terminal y escribís: 

sudo apt-get install ubuntu-gnome-desktop ubuntu-gnome-default-settings gnome-tweak-tool

Cuando nos pregunte el gestor de inicio de sesión, seleccionamos GDM. Esto instalará los paquetes básicos para usar gnome3 y el editor de configuraciones.
