Monitorear el uso de ancho de banda de tus programas con nethogs

Nethogs es una herramienta que nos permite obtener datos de la cantidad de ancho de banda que consume cada programa en tiempo real. Se encuentra en la mayoría de los repositorios, así que lo descargamos: 

# yum install nethogs

Una vez instalado correctamente lo lanzamos así: 

# nethogs wlan0
