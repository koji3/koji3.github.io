Instalando Pidora en nuestra Raspberry

Llevo mucho tiempo sin juguetear con mi Raspberry, que actualmente tiene Raspbian instalado y a la cual quiero darle un cambio de look, a ver si consigo que funcione para un servicio que tengo en mente. Así que lo primero que voy a hacer es cambiarle el sistema instalando en una SD el sistema Pidora, basado en Fedora pero adaptado a la maquinita. Lo primero que hacemos es instalar el instalador: 

sudo yum install fedora-arm-installer

Tras abrirlo podemos seleccionar la imagen si la hemos descargado o pedirle que la descargue automáticamente. Seleccionamos el dispositivo donde se instalará y procedemos con “Install“. Esto formateará el dispositivo seleccionado, así que ojito con seleccionar uno equivocado. También se puede hacer a lo old school, con el comando d**d**: 

dd if=imagen.img of=/dev/sd* bs=4M     #Cambiando el dispositivo por el correcto

Una vez acabado, introducimos la tarjeta en la Raspberry y comprobamos que funcione. Facil, no? :D
