\[ES\]Arreglar error en yum "Multilib version problems found"

Cuando estás actualizando tu distribución con yum o instalando nuevos paquetes y se interrumpe el proceso (Intencionadamente o sin querer) es probable que al intentar seguir con lo que estabas haciendo, yum te de un error parecido a este: 

Error:  Multilib version problems found.

Para arreglarlo basta ejecutar: 

sudo yum-complete-transaction
package-cleanup --cleandupes

Una vez hecho esto, prueba a volver a instalar o a acualizar. :D
