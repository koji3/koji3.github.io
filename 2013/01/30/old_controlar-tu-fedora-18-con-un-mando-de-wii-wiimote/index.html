(ES)Controlar tu Fedora 18 con un mando de Wii (WiiMote)

Hoy os voy a enseñar a controlar vuestro ordenador con un mando de Wii. No hay muchos tutoriales al respecto así que lo hago yo. Este tutorial está orientado a Fedora 18. Es posible que funcione parecido en otras distribuciones, pero no prometo nada. Para este proyecto necesitamos:

  • Un Wiimote, por supuesto
  • Un ordenador con Bluetooth (o un bichito bluetooth, que lo encuentras en Deal Extreme por menos de 3€)
  • (opcional) Nunchuck, WiiMotionPlus.
    Antes de nada instalamos lo siguiente siguiente: 
sudo yum -y install cwiid cwiid-wminput cwiid-utils

Esto nos deja que controlemos el ratón con el mando de Wii. Si no detecta la barra de infrarrojos, lo guía con los acelerómetros, pero como mola usarlo es con la barra, así que tenemos varias opciones:

  • Tener la wii encendida cerca y poner la barra encima del ordenador (método chapucero)
  • Conectar la barra a un usb, cortando y empalmando cables (con el riesgo de que nos la carguemos)
  • Fabricarnos nuestra propia barra que funcione a pilas (tutorial aquí: http://www.comolohago.cl/2009/03/08/como-hacer-una-sensor-bar-para-nintendo-wii/ ) o que funcione a través de USB (tutorial aquí: http://www.taringa.net/posts/hazlo-tu-mismo/6188059/2-Guias-para-armar-Barra-Sensora-Wii-CASERA.html)
  • Encender dos velas, una a cada lado de la pantalla (método más cutre que el primero pero que nos puede sacar de un apuro)
    Para probarlo, podemos ejecutar el comando: 
wmgui

Esto nos abre una ventana donde nos señala los botones que tenemos pulsados, tanto en el mando como en el nunchuck, los acelerómetros,… Ahora que funciona todo podemos usarlo con múltiples propósitos: mando a distancia para el reproductor de música o pelis; mando para videojuegos o para controlar la RaspberryPi a distancia (desarrollaré esto más adelante). Ahora os pongo varias configuraciones que me funcionan a mi para diversos propósitos (Estos archivos deben estar en una carpeta llamada .CWiid en tu home): Controlar VLC: 

# ~/.CWiid/VLC.conf

Wiimote.A       = KEY_SPACE     # Play/pause
Wiimote.B       = KEY_PAGEUP    # Pagina anterior

Wiimote.Up      = KEY_1         # Subir Volumen
Wiimote.Down    = KEY_2         # Bajar Volumen
Wiimote.Left    = KEY_LEFT      # Flecha izquierda
Wiimote.Right   = KEY_RIGHT     # Flecha derecha

Wiimote.Home    = KEY_M         # Silenciar
Wiimote.Minus   = KEY_R         # Activar/Desactivar aleatorio
Wiimote.Plus    = KEY_S         # Stop

Wiimote.1 = KEY_F               # Pantalla completa
Wiimote.2 = KEY_ESC             # Salir de pantalla completa

(Este archivo de configuración SOLO funciona con mi VLC, pues tengo personalizados los atajos de teclado, deberás modificarlo para que te funcione a ti también) Controlar zsnes (emulador Super nintendo) 

# ~/.CWiid/zsnes.conf

Wiimote.Up      = KEY_LEFT      # Subir Volumen
Wiimote.Down    = KEY_RIGHT     # Bajar Volumen
Wiimote.Left    = KEY_DOWN      # Flecha izquierda
Wiimote.Right   = KEY_UP        # Flecha derecha

Wiimote.Home    = KEY_ENTER     # Select
Wiimote.Minus   = KEY_F4        # Cargar Juego (Slot actual)
Wiimote.Plus    = KEY_F2        # Guardar Juego

Wiimote.1 = KEY_Z               # Salto de super mario (B)
Wiimote.2 = KEY_X               # Ataque de super mario (A)
Wiimote.A = KEY_S               # Botón X
Wiimote.B = BTN_LEFT            # Click de ratón

(Este archivo de configuración SOLO funciona con mi zsnes, pues tengo personalizados los controles, deberás modificarlo para que te funcione a ti también) Ahora, para cargar una configuración determinada ejecutamos: 

wminput -d -w -c ArchivoDeConfiguracion

Y a disfrutar!
